# ClearFeed Capture

ClearFeed Capture is a browser extension that monitors internet traffic while you browse X/Twitter, collecting data about the tweets you see for analysis by [ClearFeed](https://github.com/uchicago-capp-30320/ClearFeed).

> **Based on [Zeeschuimer](https://github.com/digitalmethodsinitiative/zeeschuimer)** by Stijn Peeters at the Digital Methods Initiative, licensed under the [Mozilla Public License 2.0](https://github.com/digitalmethodsinitiative/zeeschuimer/blob/master/LICENSE). We are grateful for their work which made this extension possible.

## Installation

Download the latest signed `.xpi` from the [releases page](https://github.com/teddykolios11/capp-zeeschuimer/releases/latest) and install it in Firefox. ClearFeed Capture only works in Firefox.

## How to use

1. Install the extension — the ClearFeed Capture icon will appear in your Firefox toolbar
2. Log into your ClearFeed account at [clearfeed.civic.garden](https://clearfeed.civic.garden) in the same Firefox window
3. Go to X/Twitter and scroll your home feed
4. The extension captures tweets automatically in the background
5. When done, click the ClearFeed Capture icon and click **"To ClearFeed"** to upload your data
6. Click **"View ClearFeed Report"** to see your feed analysis

> ⚠️ If you want to download your raw data, do so before hitting the "To ClearFeed" upload button. Tweets are cleared after each successful upload.

## Requirements

- Firefox browser
- A ClearFeed account at [clearfeed.civic.garden](https://clearfeed.civic.garden) — you must be logged in before uploading
